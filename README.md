# frontend-sidebar-dashboard
⚡ Modern • Responsive • Theme-Aware Admin Interface

✨ Project Overview

A clean and interactive Sidebar Dashboard UI built using pure HTML, CSS, and JavaScript.

This project demonstrates:

1.Modern UI/UX principles 🎨
2.Dynamic DOM manipulation ⚡
3.Theme persistence using Local Storage 💾
4.Responsive layout for all devices 📱

🔥 Core Features
1. 📂 Sidebar Navigation System
-Expand / Collapse sidebar with smooth animation
-Icon-only mode when collapsed
-Multiple toggle buttons (Navbar + Sidebar)
-Sticky sidebar for desktop layout
-Fixed overlay sidebar for mobile
2. 🌙 Dark / Light Theme System
Toggle between dark and light modes
Automatically detects system theme (prefers-color-scheme)
Saves user preference using local Storage
Dynamic icon switching (🌙 / ☀️)
Smooth toggle animation with sliding indicator
3. 🔍 Smart Search Interaction
Styled search input with icon
Auto-expands sidebar when clicked (if collapsed)
Auto-focus input for better UX
4. 📊 Navigation Menu UI
Menu items with icons + labels
Active state highlighting
Hover animations
Clean spacing and layout
5. 📱 Fully Responsive Design
Mobile sidebar overlay system
Background dim effect on small screens
Toggle navigation button for mobile
Optimized layout for tablets and phones
6. 🎨 Modern UI & Styling (CSS Highlights)
CSS Variables (Custom Properties) for theming
Light & Dark theme color systems
Smooth transitions and animations
Box shadows and rounded UI components
Scrollable sidebar with custom scrollbar
Clean typography using Google Fonts

🧠 JavaScript Functional Breakdown

1.✅ DOM Selection
Handles all UI elements:
    Sidebar
    Toggle buttons
    Theme switch
    Search form
2. ✅ Sidebar Toggle Logic
  Toggles .collapsed class
  Adjusts width and visibility
  Keeps icons visible
3. ✅ Theme Management System
      Applies saved theme from localStorage
      Detects system preference
      Updates UI dynamically
4. ✅ Theme Toggle Interaction
        Switches theme instantly
        Stores preference
        Updates toggle icon
5. ✅ Search UX Enhancement
        Detects sidebar state
        Expands automatically
        Focuses input field
