## 🚀 Features

### 📂 Sidebar Navigation System
- Expand / collapse sidebar with smooth animation  
- Icon-only mode when collapsed  
- Multiple toggle triggers (Navbar + Sidebar)  
- Sticky sidebar (desktop)  
- Overlay sidebar (mobile view)  

### 🌙 Dark / Light Theme System
- Toggle between dark & light modes  
- Detects system theme (`prefers-color-scheme`)  
- Saves user preference using `localStorage`  
- Dynamic icon switching (🌙 / ☀️)  
- Smooth toggle animation  

### 🔍 Smart Search Interaction
- Styled search input with icon  
- Auto-expands sidebar when clicked  
- Auto-focus input field for better UX  

### 📊 Navigation Menu UI
- Icon + label menu items  
- Active state highlighting  
- Smooth hover animations  
- Clean spacing & alignment  

### 📱 Responsive Design
- Mobile sidebar overlay system  
- Background dim effect  
- Toggle navigation button for small screens  
- Optimized for tablet & mobile  

### 🎨 UI & Styling Highlights
- CSS Variables (custom properties)  
- Light & dark theme color system  
- Smooth transitions & animations  
- Box shadows & rounded UI  
- Custom scrollbar  
- Google Fonts typography
